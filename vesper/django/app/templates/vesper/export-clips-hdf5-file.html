Export Clips HDF5 File

Exports clips to an HDF5 file. The clips are specified as a set of one or more detectors, one or more station/mic combinations, and a range of dates. Only clips that are at least 5205 samples long are exported, and only the first 5205 samples of those clips are exported. The HDF5 file is written to the specified path on the server. The export runs as a Vesper job. [For the time being, this command exports Call* and Noise clips. In the future, you will be able to choose the classes of clips to be exported. The clips will also be delivered via your browser rather than via the server file system.]
