Delete clip audio files

Deletes the audio files of the specified clips of this archive.

{% include "vesper/clips-specification-message.html" %}

After the audio files are deleted, whenever Vesper needs the clips' audio data it will read them from the recordings of which the clips are parts, assuming that the recordings are available. Note that if the recordings are not available, you probably don't want to run this command!
